# `Introduction to Python`
## Why Python?  
*Usually Devops Engineers works with Linux Machines.
They dont come up with GUI, so interact with these linux machines using command prompt we use shell commands.
Using SSH protocol we use Shell commands which are used to interact linux machines to do our daily tasks.*  

*If we create a file in linux system with an extension with .sh and 
write some shell commands to execute then this file would be called as shell scripting.
It will be executed as ./ filename.sh. The primary purpose of shell scripting by devops engineers is to 
intearct with linux system, to manage them and also to get some output/informaiton from the system.*  

*The Main purpose of using python is that linux wont support windows systems.So in that case python would be used.
Python can be used in both linux and windows whereas shell is only used for linux system.The foundation of 
Ansible is also python. *  

## When Python would be used?
*Python is designed to perform complex tasks, to talk to the API's,and also to do data manipulation tasks.
In python we will use some modules to perform complex tasks whereas shellscript is used to perform system and monitoring related tasks.*
